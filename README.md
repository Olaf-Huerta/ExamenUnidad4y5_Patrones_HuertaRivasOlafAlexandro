# ExamenUnidad4y5_Patrones_HuertaRivasOlafAlexandro
El siguiente es un programa el cual aplica el patrón decorador, estado, singleton y MVC, el cual muestra un menú en el cual puedes elegir plantillas de documentos (CV, folleto, certificado y factura), y cada una se puede personalizar al gusto del usuario, eligiendo entre distintas opciones que se pueden seleccionar dependiendo el formato elegido. Al final, se imprime la plantilla seleccionada con las opciones que se le agregaron.
